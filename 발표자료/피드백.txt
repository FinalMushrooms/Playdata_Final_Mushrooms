기획배경

- 정부기관에서 제공하는 공공데이터와 머신러닝을 활용하여
야생버섯에 대한 위험성과 정보를 제공하는 것을 목표

- 여러 사이트에 퍼져 있는 정보를 한 곳에서 확인 가능 
- 일러스트 등을 활용해 버섯 정보를 쉽게 전달

	- 머신러닝 CNN을 통해 버섯 이미지에 따른 속 분류
	- 딥러닝을 활용해 버섯 특징 선택시 독버섯 유무 판별
	- 공공데이터 API를 활용하여 분석된 버섯 관련 정보 제공
	
- 나아가 버섯에 관련된 요리 레시피와 사고 현황 제공


회고 
API
- 설계된 API를 활용하다보니 원하는 정보를 추출해오기 힘듬
- 페이징 기능이 API에 픽스되어 원하는 페이지를 불러오기 쉽지 않았음
- 제작 중간 API 서버 문제가 생기면 기능 전체가 마비됨

CSS
- 아직도 원하는 수준 CSS 작업이 손쉽게 안됨
- 기획당시 페이지와 제작된 페이지와의 갭이 큼
- 반응형 페이지로 설계를 못해, 노트북과 데스크탑간의 화면 간격이 큼

코딩
- 깔끔하고, 유지보수가 쉬운 코딩이 잘 안됨
- 스크립트 파일 처리 아쉬움

머신러닝
- 확률 %를 표시하고 싶었으나, 구현에 어려움을 겪음
- 속이나 버섯 형태를 정확한 한국명으로 번역이 어려움
