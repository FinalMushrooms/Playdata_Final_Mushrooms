# 사용할 기능 셋 

## 사이트 인트로?
1. 버섯 사진 여러개 들어간 박스로 독버섯 찾아보세요 선택하게 만들기
-> 선택한 버섯 정보 알려주기
-> 독버섯 구별하기 어렵다 그래서 이런 사이트 만들었다   

## 오픈 API  

1. 국명, 학명으로 버섯 검색 -> 도감 번호로 상세 검색
2. 버섯명, 목, 과, 속으로 이미지 검색 
3. 분류군, 학명, 국명 정보 검색 
4. 머신러닝으로 판별한 버섯 분류 -> 영문 이름이 나옴 -> 종류를 버섯도감 목록 검색, 분류군 정보 목록 검색, 산림생명자원 정보로 넘겨서 정보 받아오기    
5. 오픈 API로 검색 실패하면 위키피디아 또는 구글 검색으로 링크 넘기게 

## 머신러닝
1. 이미지로 판별 
    - 구글로 버섯 검색해서 나오는 이미지 넣어서 종류(속) 분류하는 기능 
    - 오픈 API 활용 검색해서 나오는 버섯 이미지를 넣어서 종류 분류하는 기능 
    - 사진파일 업로드 기능 
    
2. 특징으로 독성 판별 (텍스트 정보 기반)
    - 머리 색, 모양, 표면, 등 -> 셀렉트박스, 텍스트박스 
    - 입력 데이터 수 단계별 판별  

## 추가 기능 
1. 독버섯 또는 식용버섯 검색해서 유사한 반대 버섯 찾기
